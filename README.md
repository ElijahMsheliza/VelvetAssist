# VelvetAssist

The AI-powered booking and customer-relations assistant for independent companions.

## Features

- **Luxury Design**: Dark theme with gold accents and glassmorphism effects.
- **Interactive Demo**: Simulated chat interface showing the AI in action.
- **Waitlist Integration**: Functional signup form connected to Formspree.
- **Social Proof**: Dynamic counter showing waitlist momentum.
- **Responsive**: Fully optimized for mobile and desktop.

## Tech Stack

- **Framework**: Next.js 15 (App Router)
- **Styling**: Tailwind CSS
- **UI Library**: Shadcn/UI (custom implementation)
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Forms**: Formspree

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Deployment

### GitHub Pages
The project is configured to deploy automatically to GitHub Pages via GitHub Actions.
- **URL**: `https://[your-username].github.io/VelvetAssist/`

### Netlify
The project includes a `netlify.toml` for easy deployment.
1. Import the repository in Netlify.
2. Build command: `npm run build`
3. Publish directory: `out`
